# claude-code-workflow 프로젝트 규칙

> `/session-start` 시 자동 로딩. Claude가 반드시 준수해야 할 규칙.

## 절대 규칙 (MUST)
- 개인 경로 하드코딩 금지 (사용자별 경로 노출 주의)
- Obsidian 연동 시 경로는 사용자 환경에 맞게 설정
- Write 도구 사용 (cp 금지 — iCloud symlink 경로에서 실패)
- 모든 커맨드 파일은 영어 + 범용화 상태 유지
